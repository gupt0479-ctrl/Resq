# Design: Rescue Demo Polish

## Scope

Polish the existing hero path rather than inventing new product surfaces.

## Focus areas

- tighten stale copy that still reads like restaurant ops history
- show TinyFish mode truth where judges will look for it
- make workflow output feel like an audit log, not a generic activity feed
- keep connectors and AWS clearly secondary to the rescue scan

## Guardrails

- no broad refactor of dashboard queries or service layering
- no deep authenticated browser automation
- no heavy AWS architecture
